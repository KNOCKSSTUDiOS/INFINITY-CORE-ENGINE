# Security Policy

## INFINITY-CORE-ENGINE powered by KNOCKSSTUDiOS

## Supported Versions

| Version | Supported |
| ------- | --------- |
| Latest  | Yes       |
| < 1.0   | No        |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**DO NOT** open a public issue for security vulnerabilities.

### How to Report

1. Email: Send details to the repository owner via GitHub
2. Include: Description, steps to reproduce, potential impact
3. Response Time: We aim to respond within 48 hours
4. Fix Timeline: Critical issues addressed within 7 days

## Security Measures

- All secrets stored in environment variables (never committed)
- Dependabot enabled for automated dependency updates
- CodeQL analysis enabled for code scanning
- CODEOWNERS enforced for critical path reviews
- Branch protection on main branch

## Disclosure Policy

- We follow responsible disclosure practices
- Security patches are released as soon as possible
- Contributors are credited unless they prefer anonymity
